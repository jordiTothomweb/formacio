Exemples de Foco Interactiu

Focus natiu

Aquest és el comportament per defecte del focus natiu. No cal fer res addicional.

<button>Botó amb focus natiu</button>

Focus amb contrast insuficient

El focus no és suficient perquè coincideix amb el color del botó.

<button class="focus-insuficient">
  Botó amb focus amb contrast insuficient
</button>
    
<style>
.focus-insuficient {
    background-color: #015ECC;
    color: white;
    border: none;
}

.focus-insuficient:focus-visible {
    outline: 2px solid #015ECC;
}
</style>

Focus personalitzat

El focus s'adapta al botó i canvia de color per fer-lo més visible.

<button class="focus-personalitzat">Botó amb focus personalitzat</button>
    
<style>
.focus-personalitzat:focus-visible {
    outline: 2px solid red;
}
</style>

Focus amb espai

El focus afegeix un espai entre el contorn i el botó per millorar la visibilitat.

<button class="focus-amb-espai">Botó amb focus amb espai</button>
    
<style>
.focus-amb-espai:focus-visible {
    outline: 2px solid #1976d2;
    outline-offset: 4px;
}
</style>

Focus col·lisionant amb el fons

El focus col·lisiona amb el fons i no és prou visible.

<button class="focus-colisionant">Botó amb focus col·lisionant</button>
    
<style>
.focus-colisionant {
    background-color: #1976d2;
    color: white;
    border: none;
}

.focus-colisionant:focus-visible {
    outline: 2px solid #1976d2;
}
</style>

Focus bicolor

El focus utilitza dos colors (blanc i negre) per destacar-se clarament.

<button class="focus-bicolor">Botó amb focus bicolor</button>
    
<style>
.focus-bicolor:focus-visible {
    outline: 3px solid white;
    outline-offset: 2px;
    box-shadow: 0 0 0 4px black;
}
</style>

Focus bicolor invertit

El focus utilitza els mateixos colors que el cas anterior, però invertits.

<button class="focus-bicolor-invertit">
  Botó amb focus bicolor invertit
</button>
    
<style>
.focus-bicolor-invertit:focus-visible {
    outline: 3px solid black;
    outline-offset: 2px;
    box-shadow: 0 0 0 4px white;
}
</style>

Focus bicolor sobre botó blanc i fons fosc

El focus bicolor negre/blanc no es veu prou bé sobre un botó blanc amb fons fosc.

<button class="focus-bicolor focus-blanc">Botó blanc</button>

<style>
.focus-blanc {
  background: #fff;
  color: #222;
  border: 1px solid #bbb;
}
.focus-bicolor:focus-visible {
  outline: 3px solid black;
  outline-offset: 2px;
  box-shadow: 0 0 0 4px white;
}
</style>
