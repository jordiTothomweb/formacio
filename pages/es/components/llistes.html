Ejemplos de Listas

Ejemplo 1: Lista incorrecta

Este ejemplo muestra una lista que no utiliza elementos semánticos adecuados.

Elemento 1
Elemento 2
Elemento 3
<div>
  <div>Elemento 1</div>
  <div>Elemento 2</div>
  <div>Elemento 3</div>
</div>

Ejemplo 2: Lista correcta

Este ejemplo utiliza elementos semánticos adequats per a una llista.

  • Elemento 1
  • Elemento 2
  • Elemento 3
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ul>

Ejemplo 3: Lista con botones

Este ejemplo muestra una lista interactiva con botones.

  • Elemento 1
  • Elemento 2
  • Elemento 3 
<ul id="good-list">
  <li>
    Elemento 1
    <button>Acción 1</button>
  </li>
  <li>
    Elemento 2
    <button>Acción 2</button>
  </li>
  <li>
    Elemento 3
    <button>Acción 3</button>
  </li>
</ul>
