Ejemplos de Foco Interactivo

Focus nativo

Este es el comportamiento por defecto del focus nativo. No es necesario hacer nada adicional.

<button>Botón con focus nativo</button>

Focus con contraste insuficiente

El focus no es suficiente porque coincide con el color del botón.

<button class="focus-insuficient">
  Botón con focus con contraste insuficiente
</button>
    
<style>
.focus-insuficient {
    background-color: #015ECC;
    color: white;
    border: none;
}

.focus-insuficient:focus-visible {
    outline: 2px solid #015ECC;
}
</style>

Focus personalizado

El focus se adapta al botón y cambia de color para hacerlo más visible.

<button class="focus-personalitzat">Botón con focus personalizado</button>
    
<style>
.focus-personalitzat:focus-visible {
    outline: 2px solid red;
}
</style>

Focus con espacio

El focus añade un espacio entre el contorno y el botón para mejorar la visibilidad.

<button class="focus-amb-espai">Botón con focus con espacio</button>
    
<style>
.focus-amb-espai:focus-visible {
    outline: 2px solid #1976d2;
    outline-offset: 4px;
}
</style>

Focus colisionando con el fondo

El focus colisiona con el fondo y no es suficientemente visible.

<button class="focus-colisionant">Botón con focus colisionando</button>
    
<style>
.focus-colisionant {
    background-color: #1976d2;
    color: white;
    border: none;
}

.focus-colisionant:focus-visible {
    outline: 2px solid #1976d2;
}
</style>

Focus bicolor

El focus utiliza dos colores (blanco y negro) para destacarse claramente.

<button class="focus-bicolor">Botón con focus bicolor</button>
    
<style>
.focus-bicolor:focus-visible {
    outline: 3px solid white;
    outline-offset: 2px;
    box-shadow: 0 0 0 4px black;
}
</style>

Focus bicolor invertido

El focus utiliza los mismos colores que el caso anterior, pero invertidos.

<button class="focus-bicolor-invertit">
  Botón con focus bicolor invertido
</button>
    
<style>
.focus-bicolor-invertit:focus-visible {
    outline: 3px solid black;
    outline-offset: 2px;
    box-shadow: 0 0 0 4px white;
}
</style>

Focus con borde grueso

Un focus con un borde más grueso también puede ser suficiente para garantizar la visibilidad.

<button class="focus-gruixut">Botón con focus grueso</button>

<style>
.focus-gruixut:focus-visible {
  outline: 5px solid black;
}
</style>
