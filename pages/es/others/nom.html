Enlace con etiqueta

Inicio

El nombre accesible proviene del texto visible del enlace.

<a href="#" id="link1">Inicio</a>

Enlace con etiqueta y title

Inicio

El nombre accesible proviene del texto visible, y el `title` es un complemento.

<a href="#" title="Ir al inicio">Inicio</a>

Enlace con aria-label

El nombre accesible proviene del atributo `aria-label`. El icono es visible para todos los usuarios.

<a href="#" aria-label="Inicio">
  <i class="fas fa-home" aria-hidden="true"></i>
</a>

Enlace con aria-labelledby

Inicio texto oculto

El nombre accesible proviene del elemento con `id="descripcion_oculta_link"`. El texto es invisible pero legible para lectores de pantalla.

<span id="descripcion_oculta_link" class="visually-hidden" aria-hidden="true">
  Inicio texto oculto
</span>
<a href="#" aria-labelledby="descripcion_oculta_link">
  <i class="fas fa-home" aria-hidden="true"></i>
</a>

Enlace con aria-label y etiqueta

Inicio

Hay un conflicto entre el texto visible y el `aria-label`. Los lectores de pantalla priorizarán el `aria-label`.

<a href="#" aria-label="Enlace">Inicio</a>

Enlace con imagen con alt

El nombre accesible proviene del texto alternativo de la imagen (`alt`).

<a href="#">
  <img src="../../img/home_icon.png" alt="Inicio">
</a>

Enlace con imagen con alt vacío

Con `alt=""`, la imagen es ignorada por los lectores de pantalla. Asegúrate de que el enlace tenga otro mecanismo para proporcionar un nombre accesible.

<a href="#">
  <img src="../../img/home_icon.png" alt="">
</a>

Enlace con imagen sin alt

Sin `alt`, el lector de pantalla puede leer el nombre del archivo, lo cual no es útil.

<a href="#">
  <img src="../../img/home_icon.png">
</a>

Enlace con texto solo para lectores de pantalla

Inicio (solo lector de pantalla)

El texto Inicio (solo lector de pantalla) está oculto visualmente pero será anunciado por los lectores de pantalla gracias a la clase visually-hidden (también conocida como sr-only). 

<a href="#">
  <span class="visually-hidden">Inicio (solo lector de pantalla)</span>
  <i class="fas fa-home" aria-hidden="true"></i>
</a>

.visually-hidden {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    white-space: nowrap;
    border: 0;
}

Enlace con aria-labelledby y etiqueta visible

Nombre vía aria-labelledby Etiqueta visible

Si hay aria-labelledby y texto visible, el lector de pantalla prioriza aria-labelledby.
En este caso, se leerá: "Nombre vía aria-labelledby" 

<span id="lbl10" class="visually-hidden">
  Nombre vía aria-labelledby
</span>
<a
  href="#"
  aria-labelledby="lbl10"
>
  Etiqueta visible
</a>

Enlace con aria-label y etiqueta visible

Etiqueta visible

Si hay aria-label y texto visible, el lector de pantalla prioriza aria-label.
En este caso, se leerá: "Nombre vía aria-label" 

<a
  href="#"
  aria-label="Nombre vía aria-label"
>
  Etiqueta visible
</a>

Enlace con aria-label, aria-labelledby y etiqueta visible

Enlace con label referenciado Etiqueta visible

Cuando hay aria-label, aria-labelledby y texto visible, el lector de pantalla prioriza: aria-labelledby > aria-label > texto visible.
En este caso, se leerá: "Enlace con label referenciado" 

<span id="lbl9" >
  Enlace con label referenciado
</span>
<a
  href="#"
  aria-label="Nombre vía aria-label"
  aria-labelledby="lbl9"
>
  Etiqueta visible
</a>
