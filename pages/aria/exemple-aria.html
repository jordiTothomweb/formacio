Exemple 1: `aria-label`

Aquest exemple mostra com utilitzar l'atribut aria-label per proporcionar una etiqueta personalitzada a un element.

<button class="btn" aria-label="Desa els canvis">💾</button>

Exemple 2: `aria-labelledby`

Aquest exemple mostra com utilitzar l'atribut aria-labelledby per associar un element amb una etiqueta externa.

Formulari de contacte

<h3 id="titol-formulari">Formulari de contacte</h3> <form aria-labelledby="titol-formulari"> <label for="nom">Nom:</label> <input type="text" id="nom" /> </form>

Exemple 3: `aria-describedby`

Aquest exemple mostra com utilitzar l'atribut aria-describedby per proporcionar una descripció addicional a un element.

Introdueix un correu electrònic vàlid.
<label for="email">Correu electrònic:</label> <input type="email" id="email" aria-describedby="ajuda-email" /> <div id="ajuda-email">Introdueix un correu electrònic vàlid.</div>

Exemple 4: `aria-hidden`

Aquest exemple mostra com utilitzar l'atribut aria-hidden per amagar un element dels lectors de pantalla.

Valoració: 5 estrelles
<i class="fas fa-star" aria-hidden="true"></i> <span>Valoració: 5 estrelles</span>

Exemple 5: `aria-live`

Aquest exemple mostra com utilitzar l'atribut aria-live per anunciar canvis dinàmics en una regió de la pàgina.

<button class="btn" onclick="actualitzarMissatge()">Actualitza missatge</button> <div aria-live="polite" id="live-region" class="alert-box"></div> function actualitzarMissatge() { const liveRegion = document.getElementById('live-region'); liveRegion.textContent = 'El contingut s\'ha actualitzat correctament!'; liveRegion.hidden = false; }
