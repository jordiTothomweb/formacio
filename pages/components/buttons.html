Exemple: Divs vs Botons

Aquest exemple mostra com millorar progressivament l'accessibilitat d'un `div` i compara el resultat amb un botó natiu.

Millores Progressives amb Divs

Pas 1: Div amb aparença de botó

Div amb event click 
<div 
  class="fake-button" 
  onclick="incrementCounter('div-counter')">
  Div amb event click
</div>

Pas 2: Afegir `role="button"`

Div amb rol de botó 
<div 
  class="fake-button" 
  role="button" 
  onclick="incrementCounter('div-counter')">
  Div amb rol de botó
</div>

Pas 3: Afegir interacció amb teclat

Div amb interacció per teclat 
<div 
  class="fake-button" 
  role="button" 
  tabindex="0" 
  onclick="incrementCounter('div-counter')" 
  onkeydown="if(event.key === 'Enter' || event.key === ' ') { event.preventDefault(); this.click(); }">
  Div amb interacció per teclat
</div>

Pas 4: Botó deshabilitat

Div deshabilitat 
<div 
  class="fake-button disabled" 
  role="button" 
  tabindex="-1" 
  aria-disabled="true">
  Div deshabilitat
</div>

Comptador (Divs): 0

Botó Natiu

<button 
  onclick="incrementCounter('button-counter')"
  Botó amb event click
</button>

Comptador (Botó): 0
