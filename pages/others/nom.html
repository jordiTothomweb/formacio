Enllaç amb etiqueta i title

Inici

Enllaç amb aria-label

El nom accessible prové de l'atribut `aria-label`. La icona és visible per a tots els usuaris.

<a href="#" aria-label="Inici">
    <i class="fas fa-home" aria-hidden="true"></i>
</a>

Enllaç amb aria-labelledby

Inici text ocult

El nom accessible prové de l'element amb `id="link1"`. El text és invisible però llegible per a lectors de pantalla.

<span id="link1" class="visually-hidden" aria-hidden="true">
    Inici text ocult
</span>
<a href="#" aria-labelledby="link1">
    <i class="fas fa-home" aria-hidden="true"></i>
</a>

Enllaç amb aria-label i etiqueta

Inici

Hi ha un conflicte entre el text visible i el `aria-label`. Els lectors de pantalla prioritzaran el `aria-label`.

<a href="#" aria-label="Enllaç">Inici</a>

Enllaç amb imatge amb alt

El nom accessible prové del text alternatiu de la imatge (`alt`).

</a>

Enllaç amb imatge amb alt buit

Amb `alt=""`, la imatge és ignorada pels lectors de pantalla. Assegura't que l'enllaç tingui un altre mecanisme per proporcionar un nom accessible.
